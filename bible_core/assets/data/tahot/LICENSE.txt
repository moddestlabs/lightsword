TAHOT - Translators Amalgamated Hebrew Old Testament
===================================================

Source: STEPBible.org
License: CC BY 4.0 (Creative Commons Attribution 4.0 International)
URL: https://github.com/STEPBible/STEPBible-Data

Description:
The Leningrad Codex based on Westminster via OpenScriptures, corrected from 
color scans, with full morphological and semantic tags for all words, prefixes 
and suffixes. Semantic tags using the disambiguated Strongs is backwardly 
compatible with simple Strongs tags and includes all affixes. Morphology is 
based on ETCBC converted to the format similar to Westminster.
